Skeleton.CSS

- Mini library of custom predesigned sections, fully responsive! -

documentation

After copying the respective CDN | LINK to bring the minilibrary with the styles, choose between any of the layouts seen. Among all these, a large part uses the Flex organization system and other sections do require Grid.

Copy the HTML code of the layout that was chosen, and paste it in its respective code. It is not necessary to encapsulate it in a <div>, since each code is already packaged in a <section> tag.

It is 100% advisable not to change or remove the classes already given in the codes. If you want to make any change, it can be done from its respective CSS. In the event of a conflict between classes or styles, their styles may be interposed with the CSS: !important attribute.

<section class="fiveteen">
    <div class="container">
        <div>1</div>
        <div>2</div>
        <div>3</div>
        <div>4</div>
        <div>5</div>
        <div>6</div>
    </div>
</section>

Each section is divided by divs, these are listed to distinguish the order and positioning of each one of them. Delete the number and enter the respective text.

How can I position the divs?

I use both positioning systems, Flex and Grid, both respecting their margins and paddings, the width of the container and while it varies in different devices.

In addition to that, I used the responsive measures given by Bootstrap as a reference for other devices:

@media (min-width: 768px) { ... } @media (min-width: 576px) { ... }
